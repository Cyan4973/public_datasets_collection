# MusicBrainz Release Groups Search

Pinned MusicBrainz release-group search results for data sonification-related tags.

Pinned source: `https://musicbrainz.org/ws/2/release-group?query=tag:data&fmt=json&limit=100`

Selected series:
- `musicbrainz_score`
- `musicbrainz_count`
- `musicbrainz_release_count`
- `musicbrainz_tag_count`
- `musicbrainz_first_release_year`

Missing-value policy: Filters out release groups without a parsable first-release year.
