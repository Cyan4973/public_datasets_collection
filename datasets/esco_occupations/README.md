# ESCO Occupation Search Results

Pinned ESCO occupation search results for data-related occupations.

Pinned source: `https://ec.europa.eu/esco/api/search?type=occupation&text=data&limit=100`

Selected series:
- `esco_code_major`
- `esco_code_minor`
- `esco_preferred_label_count`
- `esco_scheme_count`
- `esco_broader_group_count`

Missing-value policy: Filters out occupations without a parsable numeric ESCO code.
