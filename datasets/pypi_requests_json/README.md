# PyPI Requests Metadata

Pinned PyPI JSON metadata for requests using release files as rows.

Pinned source: `https://pypi.org/pypi/requests/json`

Selected series:
- `pypi_file_size`
- `pypi_upload_unix`
- `pypi_yanked`

Missing-value policy: Uses release file rows; ignores top-level deprecated download counters.
