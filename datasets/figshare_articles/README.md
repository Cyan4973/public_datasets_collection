# Figshare Articles Snapshot

Pinned Figshare articles listing with article ids and type metadata.

Pinned source: `https://api.figshare.com/v2/articles?page_size=100`

Selected series:
- `figshare_article_id`
- `figshare_defined_type`
- `figshare_group_id`

Missing-value policy: Filters out article rows missing ids or publication timestamps.
