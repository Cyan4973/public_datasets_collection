# Docker Hub Repository Search

Pinned Docker Hub repository search with stars and pull counts.

Pinned source: `https://hub.docker.com/v2/search/repositories/?query=data&page_size=100`

Selected series:
- `dockerhub_star_count`
- `dockerhub_pull_count`
- `dockerhub_is_official`

Missing-value policy: Filters out repository rows missing star or pull counters.
