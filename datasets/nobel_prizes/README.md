# Nobel Prize Award Metrics

Pinned Nobel Prize API response for 2024 award-level numeric fields.

Pinned source: `https://api.nobelprize.org/2.1/nobelPrizes?nobelPrizeYear=2024`

Selected series:
- `nobel_award_year`
- `nobel_prize_amount`
- `nobel_prize_amount_adjusted`
- `nobel_laureate_count`

Missing-value policy: No missing-value substitution; rows missing prize amounts are dropped.
