# opendota_pro_matches

Public Dota 2 professional match summaries from OpenDota.

Series:
- `match_id_u64`
- `duration_u32`
- `start_time_u32`
- `radiant_team_id_u32`
- `dire_team_id_u32`
- `league_id_u32`
- `series_id_u32`
- `series_type_u8`
- `radiant_score_u16`
- `dire_score_u16`
- `radiant_win_u8`
- `version_u16`
