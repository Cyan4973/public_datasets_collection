# FreeToGame Games List

Pinned FreeToGame catalogue snapshot with ids and release dates.

Pinned source: `https://www.freetogame.com/api/games`

Selected series:
- `freetogame_game_id`
- `freetogame_release_year`
- `freetogame_release_month`
- `freetogame_release_day`

Missing-value policy: Filters out games with malformed release dates.
