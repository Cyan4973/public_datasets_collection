# NASA SDO/AIA Synoptic FITS Int16

Download-only staging recipe for SDO/AIA synoptic FITS resources discovered from the official data.gov catalog record.

Run:

```bash
staging/nasa_sdo_aia_synoptic_fits_i16/download.sh
```

The script downloads only catalog resources whose names, formats, or URLs look like FITS/archive payloads and enforces a per-file size cap.
