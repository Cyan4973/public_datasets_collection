# wger Exercises

Pinned wger exercise listing with category and relation counts.

Pinned source: `https://wger.de/api/v2/exerciseinfo/?limit=100&language=2`

Selected series:
- `wger_exercise_id`
- `wger_category_id`
- `wger_muscles_count`
- `wger_equipment_count`

Missing-value policy: Filters out exercise rows missing numeric id or category metadata.
