# iNaturalist Observations

Pinned iNaturalist observation listing with geographic coordinates and observation calendar fields.

Pinned source: `https://api.inaturalist.org/v1/observations?per_page=100&order=desc&order_by=created_at`

Selected series:
- `inaturalist_longitude`
- `inaturalist_latitude`
- `inaturalist_observed_year`
- `inaturalist_observed_month`
- `inaturalist_observed_day`

Missing-value policy: Filters out observations missing geojson coordinates or observed_on_details.
