# OpenML Dataset Metadata Snapshot

Pinned OpenML dataset listing with dataset and quality metrics.

Pinned source: `https://www.openml.org/api/v1/json/data/list/limit/100`

Selected series:
- `openml_dataset_id`
- `openml_version`
- `openml_file_id`
- `openml_number_of_features`
- `openml_number_of_instances`
- `openml_number_of_missing_values`

Missing-value policy: Filters out datasets missing the selected quality metrics.
