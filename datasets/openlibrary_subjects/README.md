# OpenLibrary Subject Works

Pinned OpenLibrary subject listing for data-related works.

Pinned source: `https://openlibrary.org/subjects/data.json?limit=100`

Selected series:
- `openlibrary_edition_count`
- `openlibrary_cover_id`
- `openlibrary_subject_count`
- `openlibrary_ia_collection_count`
- `openlibrary_author_count`
- `openlibrary_first_publish_year`

Missing-value policy: Filters out works missing cover_id or first_publish_year.
